# 牙医仪器检修小程序 - 原型说明

## 📱 项目概览

这是一个专为牙科仪器检修服务设计的微信小程序，提供报修、查询、故障自查等核心功能。

---

## 🗂️ 页面结构

### 底部导航（TabBar）

| 图标 | 标签 | 页面 | 功能描述 |
|------|------|------|----------|
| 🏠 | 首页 | `home` | 服务入口、快捷操作 |
| 🏢 | 公司 | `company` | 公司介绍、核心优势 |
| 👤 | 我的 | `mine` | 用户中心、订单管理 |

## 📦 功能模块

### 1. 立即报修（repair）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 产品名称 | 输入框 | - | 设备名称 |
| 产品型号 | 输入框 | ✅ | 设备型号 |
| 产品序列号 | 输入框 | ✅ | SN码 |
| 购买日期 | 日期选择 | - | 选填 |
| 购买凭证 | 图片上传 | - | 最多3张 |
| 故障描述 | 输入框 | ✅ | 故障详情 |
| 故障图片/视频 | 媒体上传 | ✅ | 最多3个 |
| 物流公司 | 选择 | ✅ | 顺丰等 |
| 运单号 | 输入框 | ✅ | 快递单号 |
| 寄送方式 | 单选 | - | 顺丰取件/自行寄送 |
| 收货人 | 输入框 | ✅ | 寄回信息 |
| 手机号码 | 输入框 | ✅ | 联系电话 |
| 详细地址 | 输入框 | ✅ | 寄回地址 |

### 2. 包裹查询（package-query）

```
┌─────────────────────────────────────┐
│ 【提示区】查包裹是否已到仓          │
│           输入快递单号查询状态       │
├─────────────────────────────────────┤
│ 【隐私提示】输入手机尾号查看完整轨迹 │
├─────────────────────────────────────┤
│ 【快递单号】[输入框] [扫码] [粘贴]  │
├─────────────────────────────────────┤
│ 【手机尾号】[输入框] 用于核验        │
├─────────────────────────────────────┤
│ 【立即查询】蓝色按钮                 │
├─────────────────────────────────────┤
│ 【查询结果】快递信息 · 物流进度       │
└─────────────────────────────────────┘
```

### 3. 维修进度（track）

| 状态标签 | 说明 |
|----------|------|
| 全部 | 显示所有工单 |
| 待处理 | 等待工程师接单 |
| 维修中 | 正在维修 |
| 已发货 | 已寄出 |

### 4. 故障自查（diag）

- 选择产品类型 → 选择故障类型 → 查看排查建议

### 5. 保修政策（warranty）

- 保修期限说明
- 保修范围条款
- 增值服务介绍

## 🔧 交互说明

### 扫码功能
- 包裹查询页面点击扫码图标
- 调用微信扫码API
- 自动填入快递单号

### 粘贴功能
- 包裹查询页面点击粘贴图标
- 读取剪贴板内容
- 自动填入快递单号

### 图片上传
- 支持拍照或从相册选择
- 自动压缩处理
- 支持预览和删除
