## 三、本次修复记录 (v3.0)

### 3.1 安全修补（7项 — 已完成）

| # | 漏洞 | 文件 | 修改 |
|---|------|------|------|
| C7 | `getSettings` 公开接口泄露密钥 | `cicada-client-public` | 新增16个敏感key正则黑名单自动过滤 |
| H2 | 文件上传无大小限制 (DoS) | `cicada-admin-sys/uploadGuideFile` | base64长度 + Buffer 双重 10MB 校验 |
| H3 | 图片上传仅校验扩展名 | `cicada-admin-sys/uploadSurveyPoster` | PNG/JPEG/WEBP/GIF 魔术字节校验 + 10MB限制 |
| H7 | 速率限制 IP 可被伪造 | `cicada-client-user/getClientIdentity` | 改为优先用 `getHttpInfo().clientIP` |
| C8 | Token 7天过期窗口过长 | `cicada-client-user` | `TOKEN_EXPIRE` 7天 → 48小时 |
| H6 | 包裹查询可暴力枚举手机号 | `cicada-client-order/queryPackageStatus` | 未认证用户 5分钟/20次限频 |
| H1 | verifyAdminToken 无失败限频 | `cicada-admin-sys` | IP级别 30次/5分钟冷却 |

### 3.2 架构重构（已完成）

**新建 `cicada-common` 公共模块** (291行)，统一了：

| 类别 | 统一函数 | 消除副本数 |
|------|----------|-----------|
| 数据工具 | `pickFields`, `normalizeText`, `normalizePage`, `normalizeBool`, `normalizeArray`, `normalizeIdentity`, `getFileExt` | 15→1 |
| IP 获取 | `getClientIp` (防 X-Forwarded-For 伪造) | 3→1 |
| 密码学 | `genToken`, `genSalt`, `hashPassword`, `buildPasswordFields`, `verifyPassword` | 5→1 |
| 频率限制 | `checkRateLimit`, `recordRateLimitHit`, `getRateLimitRecord`, `clearRateLimit` | 6→1 |
| Token 验证 | `verifyAdminToken` (4个版本→1个统一版), `verifyUserToken` | 6→2 |

**统一版 `verifyAdminToken` 特性**：IP限频 + 角色白名单 + 过期检查 + 禁用检查，一次调用全部覆盖。
