## 你需要完成的配置步骤

### 第 1 步：在 uniCloud 控制台开启云函数 URL 化

1. 打开浏览器，访问 [uniCloud 控制台](https://unicloud.dcloud.net.cn/)
2. 登录你的账号
3. 选择你的云服务空间
4. 点击左侧菜单"云函数/云对象"
5. 依次为以下 3 个云函数开启 URL 化：

   **cicada-admin-sys**（管理员登录、员工管理）
   - 点击函数名进入详情
   - 找到"云函数URL化"开关，点击开启
   - 复制生成的 URL（类似：`https://fc-mp-xxxxx.next.bspapp.com/http/cicada-admin-sys`）

   **cicada-admin-kb**（知识库管理）
   - 同样操作，复制 URL

   **cicada-admin-order**（工单管理）
   - 同样操作，复制 URL

### 第 2 步：更新 API 配置文件

打开文件：`E:\yayi\[03] 管理后台-Vite工程版\cicada-admin\src\config\api.js`

将第 1 步复制的 3 个 URL 分别替换到对应位置：

```javascript
export const API_BASE = {
  adminSys: '你的 cicada-admin-sys URL',
  adminKb: '你的 cicada-admin-kb URL',
  adminOrder: '你的 cicada-admin-order URL'
}
```

### 第 3 步：在小程序项目中创建管理员账号

由于 PC 端需要用管理员账号登录，你需要先在数据库中创建一个管理员账号。

**方式 1：通过 uniCloud 控制台直接插入数据**

1. 进入 uniCloud 控制台 → 云数据库 → `cicada_users` 集合
2. 点击"添加记录"，插入以下数据：

```json
{
  "openid": "admin_openid",
  "username": "admin",
  "password_hash": "你需要通过云函数生成",
  "password_salt": "你需要通过云函数生成",
  "role": "admin",
  "name": "系统管理员",
  "disabled": false,
  "create_time": 当前时间戳
}
```

**方式 2：使用维护云函数（推荐）**

小程序项目中已有 `cicada-maintenance` 云函数，可以自动将明文密码转换为哈希。

## 常见问题

**Q: 登录时提示"网络错误"？**  
A: 检查 `src/config/api.js` 中的 URL 是否正确，确保云函数已开启 URL 化。

**Q: 登录时提示"账号或密码错误"？**  
A: 确保数据库中已创建管理员账号，且密码已正确哈希。

**Q: 如何创建第一个管理员账号？**  
A: 可以临时修改 `cicada-admin-sys` 云函数，添加一个初始化接口来创建管理员。
